# 📚 EduGenie - AI Learning Assistant

EduGenie is a simple web-based AI learning assistant that allows students to ask educational questions and receive instant responses.  
It is built using **HTML, CSS, and JavaScript**.

---

## 🚀 Features
- Clean and responsive UI with modern styling
- Dropdown menu to select subjects (Python, Java, Machine Learning, AI, Data Science)
- Textarea for entering custom questions
- Interactive "Ask AI" button
- Dynamic response display with subject, question, and answer

---

## 🛠️ Technologies Used
- **HTML5** → Structure of the web page
- **CSS3** → Styling and layout
- **JavaScript (ES6)** → Logic for handling questions and generating responses

---

## ✅ Conclusion
EduGenie - AI Learning Assistant is a beginner‑friendly project that demonstrates how **HTML, CSS, and JavaScript** can be combined to build an interactive educational tool.  
It provides a clean interface, instant responses, and serves as a foundation for future enhancements such as integrating real AI APIs.  

This project highlights practical web development skills and is a strong addition to a learning portfolio.
